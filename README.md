# Bus Scheduling Optimization using MILP

## Overview
This project optimizes bus scheduling using Mixed Integer Linear Programming (MILP) to minimize cost and meet passenger demand.

## Features
- Real-world dataset (APSRTC)
- Demand modeling (peak + average)
- Time-based allocation
- MATLAB optimization (intlinprog)

## Results
- Efficient bus allocation
- Reduced cost
- Improved service quality

## Tools
- MATLAB
- Optimization (MILP)
